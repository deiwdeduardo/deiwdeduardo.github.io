Seção 2 — Ranking de Tecnologias

  1. HTML

    Base estrutural de qualquer página web, responsável pela organização do conteúdo.

  2. CSS

    Utilizado para estilizar páginas e melhorar a experiência visual do usuário.

  3. JavaScript

    Permite adicionar interatividade e comportamento dinâmico às aplicações web.

  4. React

    Biblioteca muito utilizada para construção de interfaces modernas e componentes reutilizáveis.

  5. APIs Web

    Fundamentais para integração entre sistemas e acesso a serviços externos.

Seção 3 — Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para estruturar páginas web.
CSS
Linguagem utilizada para estilização e apresentação visual.
JavaScript
Linguagem de programação voltada para interatividade na web.
DOM
Modelo de representação dos elementos de uma página HTML.
Interface que permite scripts acessarem e modificarem o conteúdo da página.
API
Conjunto de regras para comunicação entre sistemas.
Pode fornecer acesso a serviços, dados ou funcionalidades específicas.
HTTP
HyperText Transfer Protocol
Protocolo responsável pela comunicação entre clientes e servidores web.
URL
Endereço Web
Localizador que identifica recursos disponíveis na internet.
DNS
Sistema que converte nomes de domínio em endereços IP.
